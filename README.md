# Blue_prince_2D

A 2D rendition of the the Blue Prince game, just for fun !

## Requirements

- Python 3
- pip

## Install

1. Clone the deposit
    ```sh
    git clone url_de_mon_projet.git
    ```

2. Create a virtual environnement

3. Install the dependencies
    ```sh
    pip install -r requirements.txt
    ```


## How to use

From src/ :

```sh
python main.py
```

## Debugg

From src/ :

```sh
python tests/test_UI.py
```
