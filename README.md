# Blue_prince_2D

A 2D rendition of the the Blue Prince game, just for fun !

## Requirements
- Python 3
- pip

## Install
1. Clone the deposit
    ```sh
    git clone url_de_mon_projet.git
    ```
2. Create a virtual environnement

3. Install the dependencies
    ```sh
    pip install -r requirements.txt
    ```


## Execute
Set cwd to ```src/``` then run:
```sh
python main.py
```
Or from any cwd :
- you may create a shortcut to src/main.py
- you can run file (with python)  : ```blueprince.py``` of Project root directory.\
It is a pseudo os-independant shortcut to ./src/main.py (automatically sets cwd)

## Debugg
Set cwd to ```src/``` then run:
```sh
python ../tests/test_UI.py
```
Or from cwd at ```tests/```, run:
```sh
python test_UI.py
```
